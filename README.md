# atcoder-julia
JULIA_VERSION = 1.4.0  
インストール済みライブラリ: DataStructures, Primes

## Usage
1. Junoを開き，`Shift + Command + P` で `Julia Client: New Terminal` を選択
1. 以下を入力
```sh
$ make build
$ make run
```
3. `ATOM_PORT`を聞かれるので，`Shift + Command + P` で `Julia Client: Connect External Proccess` を選択し，出てくる数字を送信
